2. Activity 2 Smart Building:

In this activity, the concept smart building is explained. Smart building has intelligent networking and automation, independent actions and ability to learn with the features such as multifunctionality, safety and security. It would have places with different climate, and maintain temperature in accordance to the incoming real time data.

3. Activity 3 Control Interface:

In this activity, the concept of control interface is understood. It's main features are as ecoefficiency, interactive UI, high security and the parts like sensors,user interface and cloud software. The users would be residents, employees and visitors.
